## üöÄ QUICK START

### 1-Minute Setup
```bash
# Clone/navigate to project
cd carwash-00

# Install everything
npm run install-all

# Seed database with test data
cd apps/api && node seed.js && cd ../..

# Start all services (3 terminals)
npm run api      # Terminal 1 - Backend
npm run web      # Terminal 2 - Web App
npm run mobile   # Terminal 3 - Mobile App
```

### Access the System
- **Web App**: http://localhost:5173
- **Mobile App**: Scan QR code with Expo Go
- **API**: http://localhost:3000
- **Admin Login**: admin@carwash.com / admin123
